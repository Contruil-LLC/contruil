# Contruil

**Human-Governed AI Orchestration**

Enterprise AI governance layer providing multi-model orchestration with intent-based routing, four-gate validation, and tamper-evident audit trails.

**U.S. Provisional Patent Application No. 63/980,310 — Patent Pending**

## Stack

- [Next.js](https://nextjs.org/) 14
- [Tailwind CSS](https://tailwindcss.com/) 3.4
- [TypeScript](https://www.typescriptlang.org/) 5.4
- Deployed on [Vercel](https://vercel.com)

## Development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deployment

Connected to Vercel. Push to `main` to deploy.

- **Production domain:** contruil.com
- **Framework:** Next.js (auto-detected by Vercel)
